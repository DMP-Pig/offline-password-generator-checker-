offline-password-generator-checker 离线密码生成器与安全检查器
简体中文
项目介绍

offline-password-generator-checker 是一款离线密码工具，集「高强度密码生成」与「密码强度检测」于一体，无需联网、不上传任何数据，所有操作均在本地浏览器完成，彻底保护你的隐私安全。

适合日常注册网站、设置账号密码时使用，既能快速生成安全可靠的密码，也能检测自己现有密码的安全等级，避免使用弱密码导致账号泄露风险，轻量、实用、无任何外部依赖。

核心功能

🔒 密码生成功能

- 自定义密码长度（支持 6-64 位，默认 16 位，满足不同平台密码长度要求）

- 可自由选择密码组成：大写字母、小写字母、数字、特殊字符（!@#$%^&*()_+-=[]{}|;:,.?~`），按需组合

- 一键生成密码，支持点击复制到剪贴板，复制成功提示，操作便捷

- 生成密码后自动关联强度检测，实时显示当前生成密码的安全等级

🛡️ 密码强度检测功能

- 支持手动输入自定义密码，实时检测并显示密码强度（弱 / 中 / 强 / 极强）

- 检测逻辑贴合实际安全需求，综合评估密码长度、字符多样性、复杂度，拒绝“伪强密码”

- 强度可视化展示（进度条 + 文字提示），直观区分密码安全等级

✅ 离线隐私保护

- 所有密码生成、检测操作均在本地浏览器运行，不收集、不上传任何用户数据，隐私零泄露

- 支持本地保存（可选，基于 localStorage）常用密码生成配置（如默认长度、常用字符组合），仅保存在本地，不同步至任何服务器

快速使用

1. 下载本项目全部文件（clone 仓库或下载 ZIP 压缩包并解压）

2. 找到解压后的 index.html 文件

3. 双击 index.html，用任意浏览器（Chrome、Edge、Firefox 等）打开即可使用，无需联网

使用步骤演示

1. 生成密码

- 打开工具后，在「密码生成」区域，设置所需密码长度

- 勾选需要包含的字符类型（如同时勾选大写、小写、数字、特殊字符，生成密码强度最高）

- 点击「生成密码」按钮，下方将显示生成的密码

- 点击「复制密码」按钮，提示“复制成功”后，即可粘贴到需要设置密码的地方

2. 检测密码强度

- 在「密码强度检测」区域，手动输入需要检测的密码

- 输入过程中，工具将实时更新强度进度条和文字提示，判断密码为弱 / 中 / 强 / 极强

- 根据提示调整密码（如增加长度、添加特殊字符），提升密码安全等级
